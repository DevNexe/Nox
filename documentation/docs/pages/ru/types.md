# Типы

Базовые типы: числа, строки, булевы значения, списки, кортежи, словари, множества и none.

```
n = 42
pi = 3.14
name = "nox"
flag = true
items = [1, 2, 3]
point = (1, 2)
map = {"a": 1}
setv = {1, 2, 3}
empty = none
```

## Тип None

Тип `none` представляет отсутствие значения. Аналог `null` в других языках.

```
x = none

if x == none:
    display("x это none")

if x != none:
    display("x имеет значение")

define get_optional():
    result none
```

## Индексация

Доступ к отдельным элементам через квадратные скобки:

```
items[0]          # Получить элемент с индексом 0
items[1] = 42     # Установить элемент с индексом 1
```

## Срезы строк (Slicing)

Извлечение подстрок с использованием Python-стиля нотации `[start:stop:step]`.

```
text = "Nox"

# Базовые срезы
text[0:3]         # "Nox" - с индекса 0 по 3
text[3:]          # "Lang" - с индекса 3 до конца
text[:3]          # "Nox" - с начала до индекса 3
text[:]           # "Nox" - вся строка (копия)

# Отрицательные индексы
text[-1]          # "g" - последний символ
text[-4:]         # "Lang" - последние 4 символа
text[:-1]         # "NoxLan" - все кроме последнего

# Шаг/stride
text[::2]         # "NxLn" - каждый 2-й символ
text[::-1]        # "gnaLxoN" - в обратном порядке

# Сложные срезы
text[1:6:2]       # "oL" - с индекса 1 по 6, каждый 2-й
```

Срезы работают с любым типом последовательности: строки, списки, кортежи.

```
items = [1, 2, 3, 4, 5]
items[1:4]        # [2, 3, 4]
items[::2]        # [1, 3, 5]
items[::-1]       # [5, 4, 3, 2, 1]

tuple_data = (10, 20, 30, 40)
tuple_data[1:3]   # (20, 30)
```
