# Синтаксис

## Блоки

Блоки задаются отступами (только пробелы). Табуляции запрещены.

```
if x > 0:
    display("positive")
else:
    display("non-positive")
```

## Многострочные структуры

Код внутри скобок `()`, `[]`, `{}` может занимать несколько строк. Отступы внутри скобок игнорируются.

```
# Многострочный список
items = [
    1,
    2,
    3,
    4,
    5
]

# Многострочный словарь
config = {
    "host": "localhost",
    "port": 8080,
    "features": ["api", "web"]
}

# Многострочный вызов функции
result = process(
    arg1,
    arg2,
    arg3
)

# Многострочный кортеж
coordinates = (
    x,
    y,
    z
)
```

## Оператор pass

Оператор `pass` — это нулевая операция; при выполнении ничего не происходит. Полезен как заполнитель, когда оператор требуется синтаксически, но код не нужно выполнять.

```
if condition:
    pass
else:
    display("условие ложно")

define placeholder_function():
    pass
```
