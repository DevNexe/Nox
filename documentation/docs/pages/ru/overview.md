# Язык Nox

Nox — скриптовый язык с синтаксисом в стиле Python, явными блоками и компактной стандартной библиотекой. Эта вики описывает текущий синтаксис и поведение рантайма.

## Быстрый пример

```
define add(a, b):
    result a + b

x = add(2, 3)
display("sum", x)
```

Ключевые идеи: отступы для блоков, `define` для функций, `result` для возврата, `display` для вывода.

## Основные возможности

### Базовые конструкции языка
- **Блоки с отступами** - Синтаксис наподобие Python с использованием пробелов
- **Переменные и типы** - Динамическая типизация с числами, строками, списками, словарями, кортежами, множествами
- **Функции** - Определяются с помощью `define`, возвращают значения через `result`
- **Управление потоком** - `if/else`, `for`, `while`, `match/case`
- **Оператор pass** - Заполнитель для пустых блоков
- **Значение none** - Представляет отсутствие значения

### Возможности структур данных
- **Многострочные структуры** - Списки, словари, кортежи и вызовы функций могут занимать несколько строк
- **Срезы строк** - Python-стиля нотация срезов `[start:stop:step]`
- **Индексация и доступ** - Доступ к элементам через `[индекс]` или `.атрибут`
- **Коллекции** - Списки, словари, кортежи, множества и строки

### Продвинутые возможности
- **Лямбда-выражения** - Встроенные анонимные функции
- **Декораторы** - Декорирование функций с помощью `@`
- **Классы и ООП** - Объектно-ориентированное программирование с наследованием
- **Traits** - Похожие на интерфейсы контракты для классов
- **Обработка исключений** - Блоки try/except/finally
- **Async/await** - Поддержка асинхронных функций
- **Сопоставление по образцу** - Оператор `match` для сложных условий

## Пример с новыми возможностями

```
# Многострочный словарь
config = {
    "database": "localhost",
    "port": 5432,
    "options": [
        "ssl",
        "compression"
    ]
}

# Срезы строк
url = "https://example.com"
host = url[8:]          # "example.com"

# Функция с многострочными аргументами
define process(data, options, verbose):
    if verbose == none:
        pass
    else:
        display(verbose)
    result data

result = process(
    config,
    ["prod"],
    none
)
```
