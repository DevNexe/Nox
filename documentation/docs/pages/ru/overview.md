# Язык Nox

Nox — скриптовый язык с синтаксисом в стиле Python, явными блоками и компактной стандартной библиотекой. Эта вики описывает текущий синтаксис и поведение рантайма.

## Быстрый пример

```
define add(a, b):
    result a + b

x = add(2, 3)
display("sum", x)
```

Ключевые идеи: отступы для блоков, `define` для функций, `result` для возврата, `display` для вывода.
