# Оператор With

`with expr as name:` связывает значение с именем `name` внутри блока.

Если у значения есть `close()`, Nox вызовет его автоматически при выходе из блока.

```
with open("log.txt", "r") as f:
    text = f.read()
    display(text)
```

Это удобно для файлов и других закрываемых ресурсов.
