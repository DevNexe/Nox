# Многострочные структуры

Код внутри скобок `()`, `[]`, `{}` может занимать несколько строк. Отступы игнорируются внутри скобок.

Эта возможность облегчает написание читаемого и хорошо отформатированного кода для сложных структур данных и вызовов функций.

## Многострочные списки

```
items = [
    1,
    2,
    3,
    4,
    5
]
```

## Многострочные словари

```
config = {
    "host": "localhost",
    "port": 8080,
    "features": ["api", "web"]
}

# Вложенная многострочная структура
database_config = {
    "primary": {
        "host": "db1.example.com",
        "port": 5432
    },
    "replica": {
        "host": "db2.example.com",
        "port": 5432
    }
}
```

## Многострочные вызовы функций

```
result = process(
    arg1,
    arg2,
    arg3
)

data = complex_calculation(
    value1,
    value2,
    options={
        "timeout": 30,
        "retry": 3
    }
)
```

## Многострочные кортежи

```
coordinates = (
    x,
    y,
    z
)
```

## Смешивание многострочных структур

Структуры могут быть вложены и смешаны:

```
response = {
    "status": 200,
    "data": [
        {"id": 1, "name": "Alice"},
        {"id": 2, "name": "Bob"}
    ],
    "callback": lambda x: process(
        x,
        verbose=true
    )
}
```

## Правила

- Переносы строк внутри `()`, `[]`, `{}` не требуют специального синтаксиса
- Отступы игнорируются внутри скобок
- Финальные запятые разрешены
- Комментарии работают внутри многострочных структур
