# Блоки

Блоки кода определяются за счет отступов (только пробелы). Табуляция запрещена.

Блоки используются в управлении потоком выполнения, определениях функций и классов.

## Базовая структура блока

```
if x > 0:
    display("положительное")
else:
    display("не положительное")
```

## Вложенные блоки

Блоки могут быть вложены на любую глубину:

```
if условие1:
    if условие2:
        display("оба верны")
    else:
        display("условие2 ложно")
else:
    display("условие1 ложно")
```

## Правила отступов

- Используйте только пробелы (рекомендуется 4 пробела)
- Табуляция **запрещена** и вызовет ошибку
- Все строки с одинаковым отступом принадлежат одному блоку
- Отступы должны быть последовательными во всем файле
