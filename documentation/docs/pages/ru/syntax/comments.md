# Комментарии

Комментарии используются для объяснения кода и игнорируются при выполнении.

## Однострочные комментарии

```
# Это комментарий
x = 5  # Установить x на 5

# Несколько комментариев
# на отдельных строках
# это нормально
```

## Стиль комментариев

- Комментарии начинаются с `#` и продолжаются до конца строки
- Используйте комментарии для объяснения **почему**, а не только что
- Держите комментарии краткими и ясными
