# Установка и настройка

Быстро начните работу с Nox.

## Предварительные требования

- Python 3.8 или выше
- Текстовый редактор или IDE
- Доступ к командной строке

## Установка

1. Клонируйте или скачайте репозиторий Nox
2. Перейдите в каталог проекта
3. Внешние зависимости не требуются

## Запуск программ Nox

### Из командной строки

```bash
python -m nox your_program.nox
```

### Пример файла: hello.nox

```
display("Привет, Nox!")
```

Запустите его:

```bash
python -m nox hello.nox
```

## Ваша первая программа

Создайте `first_program.nox`:

```
# Дисплей сообщений
display("Добро пожаловать в Nox")

# Переменные
name = "Alice"
age = 30

# Вычисления
total = age * 2

# Дисплей результатов
display("Имя: " + name)
display("Возраст: " + age)
display("Двойной возраст: " + total)
```

Запустите его:

```bash
python -m nox first_program.nox
```

Вывод:

```
Добро пожаловать в Nox
Имя: Alice
Возраст: 30
Двойной возраст: 60
```

## Поиск неисправностей

- **"command not found"**: Убедитесь, что Python установлен и в PATH
- **"No such file"**: Проверьте, что файл .nox существует в текущем каталоге
- **Ошибка синтаксиса**: Просмотрите документацию синтаксиса для правильной структуры
