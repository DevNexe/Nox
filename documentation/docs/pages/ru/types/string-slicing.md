# Срезы строк

Срез строки извлекает часть строки, используя синтаксис `string[start:stop:step]`.

## Базовые срезы

Извлечь подстроку с `start` до `stop`:

```
text = "Привет, Мир!"

display(text[0:6])    # "Привет"
display(text[8:12])   # "Мир!"
display(text[0:1])    # "П"
```

## Срезы без начала или конца

Пропустите `start` чтобы начать с начала, или пропустите `stop` чтобы дойти до конца:

```
word = "Python"

display(word[:4])     # "Pyth"
display(word[2:])     # "thon"
display(word[:])      # "Python" (вся строка)
```

## Отрицательные индексы

Отрицательные индексы считаются с конца:

```
text = "Nox"
display(text[-4:])    # "Lang"
display(text[:-2])    # "NoxLa"
display(text[-5:-2])  # "Lan"
```

## Параметр шага

Шаг определяет направление и интервал:

```
text = "0123456789"

display(text[0:10:2])   # "02468"
display(text[1:8:2])    # "1357"
display(text[::-1])     # "9876543210" (обратно)
display(text[5:1:-1])   # "5432" (назад, перед 1)
```

## Примеры

```
# Каждый второй символ
code = "Python"
display(code[::2])      # "Pto"

# Последние 3 символа
url = "example.com"
display(url[-3:])       # "com"

# Развернуть строку
word = "level"
reversed_word = word[::-1]
display(reversed_word)  # "level"

# Пропустить первый и последний символ
name = "Alice"
middle = name[1:-1]
display(middle)         # "lic"
```

## Правила

- `[start:stop]` не включает символ в позиции `stop`
- Индексы могут быть отрицательными (считаются с конца)
- `step` может быть отрицательным (обход в обратном направлении)
- Индексы за пределами границ не вызывают ошибку
